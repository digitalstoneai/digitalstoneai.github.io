PROJECT: InspectAgent — AI-Assisted Infrastructure Inspection System AGENCY: DigitalStone AI VERTICAL: Infrastructure / Defense Adjacent / Government Contracting CLIENT: Pacific Infrastructure Services, San Diego CA (CalTrans + Port of SD contracts) CHALLENGE: Manual drone/ROV footage review taking 6–8 hours per structure. Inconsistent reporting across inspectors. No audit trail for defect classification decisions. SYSTEM & STACK: - Footage ingestion: ffmpeg frame extraction (1 frame/second for standard, 4fps for flagged zones) - Vision model: OpenAI GPT-4V for defect classification + severity scoring (0–5 scale) - Defect taxonomy: 14 defect types (cracking, spalling, corrosion, rebar exposure, water intrusion, efflorescence, joint failure, bearing wear, scour, delamination, impact damage, paint failure, debris, biological growth) - Database: PostgreSQL (frame ID, location coordinates, defect type, severity, confidence score, inspector override) - Report agent: LangChain → structured PDF with defect map, priority list, recommended actions - Workflow: n8n orchestration - Archive: Google Cloud Storage (footage + extracted frames + reports) - Human review: inspector sees AI flags, confirms or overrides — all decisions logged WORKFLOW: Footage upload → Frame extraction → Vision classification → Defect DB entry → Severity scoring → Priority ranking → Report generation → Inspector review → Final PDF delivery DEFECT CLASSIFICATION RESULTS (82-frame test set): - Exact severity match: 91% (75/82 frames) - Off-by-one severity: 7% (acceptable tolerance) - Over-flagged (false positive): 1.2% - Under-flagged (missed defect): 0.8% — all minor severity - Critical defect detection rate: 100% (zero misses) STRUCTURE RESULTS (12 bridge/pier inspections): - Avg review time: 1.4 hrs (down from 6–8 hrs) - Report consistency: 100% standardized format - Inspector override rate: 9% (system learns from overrides) - Structures flagged for immediate action: 3 of 12 (rebar exposure, severe corrosion) RELEVANCE TO DEFENSE/GOV CONTRACTING: This system architecture is directly applicable to: - NIWC Pacific facility inspection workflows - Port infrastructure assessment for Navy/Coast Guard - ROV-based subsea inspection (pipeline, pier, hull) - Bridge inventory programs under FHWA requirements CLIENT QUOTE: "What used to take my best inspector a full day now takes ninety minutes — and the report is cleaner and more consistent than anything we were producing manually." — Operations Director, Pacific Infrastructure Services
